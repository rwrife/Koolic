09 — Segmented (piecewise) easing

Stitch multiple easings together across the timeline. Each segment is { to: <fraction in (0,1]>, easing: 'name' | function }, and segments must end at to: 1. 

$$(model).x.animate(0, 720, 2500, [
  { to: 0.20, easing: 'easeInSine'  },
  { to: 0.80, easing: 'linear'      },
  { to: 1.00, easing: 'easeOutSine' }
]);
linear (reference)
slow-in / linear / slow-out
