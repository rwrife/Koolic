About this app

Each top-level page (home.html, about.html, blog.html, contact.html) is a standalone HTML file. The router fetches them with the browser's fetch API, parses out the <body>, and swaps it into the shell's mount point.

What works

  • Inline <script> tags inside fetched pages are re-executed. See the time stamp below — it was set by a script inside this page:

What doesn't

  • Module scripts (type="module") are not re-executed — they dedupe by URL.
  • Don't reference document.body or expect the host page to reload. Use the enter hook (declared in the route config) for navigation-aware setup.
