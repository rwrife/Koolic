Welcome

This is a real, separate HTML file — open examples/14-multi-page/pages/home.html in your editor. It contains no shell, no scripts to wire up, no framework boilerplate.

The shell (header, footer, nav) lives in index.html. The router fetches this file's <body> contents when you visit #/ and drops it into the shell's #app container. No page reload.

Try it: open DevTools' Network tab and click around. You'll see about.html, blog.html, etc. arrive one at a time — and only once each, because the router caches them.
