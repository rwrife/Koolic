Welcome

A small single-page app built with Koolic's router, $$ bindings, and the memory-backed state store.

Try clicking around — note how:

  • The URL changes (look at the address bar's #/…) without a full page reload.
  • The cart count in the header stays consistent because it's bound to a shared store.
  • The back button and refresh both Just Work.
  • Unknown URLs (try #/whatever) fall through to a 404.

About

This demo combines three Koolic libraries:

  • koolic.js — reactive properties, DOM binding
  • koolic.store.js — the cart lives in a memory store; it survives navigation but not full reloads
  • koolic.router.js — view switching, :id parameters, click delegation, a 404 fallback

Each <section data-view> in the markup is one "page". The router shows the one whose path matches and hides the rest. Cart count badge, page title, and breadcrumb are all reactive bindings — no manual DOM updates anywhere.
